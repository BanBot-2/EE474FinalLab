◆ buttonInterrupt()

void IRAM_ATTR buttonInterrupt ( )

Interrupt Service Routine (ISR) triggered by a change on the button pin (BUTTON_PIN).

This ISR implements debouncing logic using a hardware timer. If the button state change is stable, it notifies the xTaskLCD to handle the alarm enable/disable toggle or mode change.

◆ vTaskIR()

void vTaskIR ( void * pvParameters)

FreeRTOS Task to handle the Infrared (IR) remote input.

This task processes notifications from the remoteInterrupt and implements an IR debouncing mechanism. If a valid IR pulse count is detected (indicating a remote button press), it sends a notification to xTaskLCD, simulating a button press (e.g., for snooze/confirm).

Parameters
pvParametersNot used.

◆ vTaskLCD()

void vTaskLCD ( void * pvParameters)

FreeRTOS Task for managing the Liquid Crystal Display (LCD) and handling alarm setting logic.

This task is responsible for:

  1. Reading ambient light from xPhotoQueue to control the LCD backlight.
  2. Displaying the current time and temperature.
  3. Toggling the alarm state based on button input.
  4. Transitioning to alarm setting modes (hour and minute) based on potentiometer movement.
  5. Reading potentiometer input to set the alarm hour and minute, and sending the new time via xAlarmQueue.
Parameters
pvParametersNot used.

◆ vTaskPhotoPot()

void vTaskPhotoPot ( void * pvParameters)

FreeRTOS Task for reading the Photoreistor (LDR) and Potentiometer (POT) analog values.

This task reads the analog inputs from POT_PIN and PHOTO_PIN and sends the values to their respective queues (xPotQueue, xPhotoQueue) for other tasks to use.

Parameters
pvParametersNot used.

◆ vTaskRTC()

void vTaskRTC ( void * pvParameters)

FreeRTOS Task for managing the Real-Time Clock (RTC) module.

This task is responsible for:

  1. Periodically reading the current time and temperature from the RTC and updating currentTime and currentTemp (protected by xTimeTempMutex).
  2. Receiving new alarm times from xAlarmQueue and configuring the RTC alarm.
  3. Handling notifications from xTaskLCD to enable/disable the RTC alarm setting.
  4. Checking if the RTC alarm has fired and starting/stopping the physical alarm (buzzer and LED timers).
Parameters
pvParametersNot used.
